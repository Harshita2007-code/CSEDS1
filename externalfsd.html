MODULE DETAILS

description list

HTML
HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages.
HTML describes the structure of a Web page.
HTML consists of a series of elements.
HTML elements tell the browser how to display the content.
CSS styling
CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in a markup language, most commonly HTML.
It controls the visual appearance of web pages, including elements like colors, fonts, spacing, layout, and responsiveness.
Javascript fundamental
JavaScript is a dynamic programming language used for creating interactive, dynamic web pages and applications.
It is a core technology of the web, working alongside HTML (for content structure) and CSS (for styling) to bring web experiences to life.
